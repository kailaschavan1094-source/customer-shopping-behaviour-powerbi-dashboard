# Customer Shopping Behaviour Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes customer shopping behaviour using interactive KPIs, charts, and slicers. It helps understand customer preferences, purchasing patterns, and business performance.

## 📂 Dataset
- Customer Shopping Behaviour Dataset
- Cleaned CSV Dataset
- Power BI (.pbix) File

## 📊 KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Age
- Average Rating
- Average Purchase

## 📈 Dashboard Visuals

### Page 1 – Executive Dashboard
- KPI Cards
- Sales by Category (Donut Chart)
- Sales Trend (Line Chart)
- Sales by Location (Map)
- Top Sales by Brand (Bar Chart)

### Page 2 – Customer Purchase Insights
- Average Purchase by Festival/Sale
- Purchase Amount by Gender & Category
- Average Purchase by Age (Scatter Chart)
- Customers by Color (Donut Chart)
- Purchase Amount by Brand (Treemap)

## 🎯 Features
- Interactive Slicers
- Dynamic Filtering
- Business Insights
- Clean Dashboard Design

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset
